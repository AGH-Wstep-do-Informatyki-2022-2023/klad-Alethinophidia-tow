#### **Główne** cele gry:


1. Zbieramy punkty.
2. Walczymy z przeciwnikami
    1. Robimy to w dobrym momencie
    2. Uważamy, żeby nie zginąć ani nie dać się zablokować.
3. Nie dajemy się złapać.
